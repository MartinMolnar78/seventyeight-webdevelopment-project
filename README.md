# SEVENTYEIGHT – Moto blog (Záverečný projekt, 2022) / Moto Blog (Final Project, 2022)

## 🇸🇰 Slovensky
Responzívna webová stránka vytvorená ako záverečný projekt z predmetu **Jazyky webu** (2022).  
Ide o jednoduchý osobný blog o prestavbe mojej motorky, postavený na **HTML/CSS/JavaScript** s využitím **Bootstrap 5** komponentov.

### Funkcie
- Viac stránok prepojených navigáciou (responzívny navbar + hamburger menu na mobiloch)
- Fixná navigácia počas scrollovania (`fixed-top`)
- Galérie s **Bootstrap Carousel** (slideshow)
- **Accordion** sekcie
- **Tabuľka** a **zoznamy** v obsahu
- Kontaktný formulár s validáciou (kontrola e-mailu, povinný súhlas) + redirect na `thankyou.html`
- Jednoduchý JavaScript nástroj: **Premix kalkulačka** (výpočet oleja podľa litrov benzínu a pomeru miešania)

### Technológie
- HTML5, CSS3, JavaScript
- Bootstrap 5 (CDN)

### Stránky (príklady)
- `index.html` – hlavná stránka
- `blog.html` – domov / úvod
- `galeria.html` – galéria
- `formular.html` – formulár
- `premix.html` – premix kalkulačka
- `thankyou.html` – potvrdenie po odoslaní formulára
- `project*.html` – články/projekty

### Štruktúra repozitára
Projekt je rozdelený do priečinkov `project/` a `screenshots/`.

### Spustenie
Najjednoduchšie je spustiť projekt cez **Visual Studio Code**:
1. Otvor priečinok repozitára vo VS Code
2. Nainštaluj rozšírenie **Live Server**
3. Otvor `project/index.html` a klikni **Go Live**

## 🇬🇧 English
Responsive multi-page website built as a final project for the **Web Languages** course (2022).  
It’s a simple personal blog about rebuilding my motorcycle, built with **HTML/CSS/JavaScript** and **Bootstrap 5** components.

### Features
- Multi-page navigation (responsive navbar + mobile hamburger menu)
- Sticky/fixed header while scrolling (`fixed-top`)
- Galleries using **Bootstrap Carousel** (slideshow)
- **Accordion** sections
- Content with **tables** and **lists**
- Contact form with validation (email check, required consent checkbox) + redirect to `thankyou.html`
- Simple JavaScript tool: **Premix calculator** (oil amount based on fuel liters and mixing ratio)

### Tech Stack
- HTML5, CSS3, JavaScript
- Bootstrap 5 (CDN)

### Pages (examples)
- `index.html` – landing page
- `blog.html` – home / intro
- `galeria.html` – gallery
- `formular.html` – contact form
- `premix.html` – premix calculator
- `thankyou.html` – form submission confirmation
- `project*.html` – articles/projects

### Repository Structure
The repo is split into `project/` and `screenshots/`.

### Run locally
The easiest way to run the project is with **Visual Studio Code**:
1. Open the repository folder in VS Code
2. Install the **Live Server** extension
3. Open `project/index.html` and click **Go Live**
